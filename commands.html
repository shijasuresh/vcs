Git Commands

  • git --version: This gives the current version of the git on the system
  • git init: This initialises the git for the current project
  • git status: This gives the status of the files. If a file is new, it will show it with 'U' (untracked) new file and if the file is modified, it is shown with 'M' (modified)
  • git add .: This will add all the files with changes to be staged
  • git add filename: This will add only the mentioned file to be staged
  • git commit -m 'msg': This will commit the changes with the message mentioned in the single quotes
  • git log: This will show the log of the commits made
  • git reset --hard versionfromlog: This will revert the changes to the mentioned version and also clear/delete the code
  • git reset --soft versionfromlog: This will revert the changes to the mentioned version but will retain the code


To use repository on github

  1. Create the repository on github
  2. Open Cmd/Terminal
  3. Navigate to the desired folder
  4. git remote add origin linkofgithubrepo: This will create the link to the remote github repository
  5. git branch -M main: This will create the main branch inside the repository
  6. git push -u origin main: This is used to push the code onto the remote repository the very first time
  7. After this, we can use the following commands for future versioning
  8. git add .: Add the files to be changed
  9. git commit -m 'msg': This will commit the changes with the message
  10. git push: We need to write only push for future changes


Cloning a repository

  1. Open Cmd/Terminal
  2. Navigate to the folder you wish to store the clone in
  3. git clone linktoclonefrom: This will clone the repository from the github link
  4. git pull: Navigate to the folder. This will help to pull any changes that is made to the original repository. If no changes are made, it will inform that the folder is 'Already up to date'


Branching

  • git branch: This will show the list of branches and the branch currently in use
  • git branch branchname: This will create a new branch with name branchname
  • git checkout branchname: This will switch from the current branch to branchname
  • git merge branchname: Navigate to the branch you need to merge into and then use this command to merge branchname branch
